pyInkscape APIs
===============

The most important class in pyInkscape is the :class:`pyinkscape.Canvas`.
This represents an Inkscape canvas (i.e. SVG file).

Graphical objects (text, circle, rectangle, etc.) can be drawn onto available :class:`pyinkscape.inkscape.Group` objects.
